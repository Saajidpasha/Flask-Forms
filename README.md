# Flask-Forms

INSTRUCTIONS:
Download/Clone all the files and navigate to the directory of the downloaded files via terminal -->Example :  cd yourfolderlocation
Run python myformapp.py
You should be able to see a contact form

REQUIREMENTS:
1.Python 3 or above
2.Flask

Installations:
pip3 install flask
